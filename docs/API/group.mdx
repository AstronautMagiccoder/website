Base URL: `https://api.romanager.jaydensar.net/v1`

## Role
`PATCH /role/:user`

User parameter is a Roblox account ID.
### Request body
*Only one of these parameters are required in the request body*.
```json
{
	"roleId": // the target role's ID
	"roleRank": // the target role's rank
	"roleName": // the target role's name
}
```

## Exile
`DELETE /exile/:user`

User parameter is a Roblox account ID.
### Request body
None.

## Shout
`PATCH /shout`
### Request body
```json
{
	"content": // the shout's content
}
```
