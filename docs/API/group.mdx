Base URL: `https://api.romanager.jaydensar.net/v1`

## Assign a role to a user

```http
PATCH /role/:userId
```
*Only one of these parameters are required*
| Parameter | Type     | Description            |
| :-------- | :------- | :--------------------- |
| `roleId`  | `number` | ID of role to assign   | 
| `roleRank`| `number` | Rank of role to assign |
| `roleName`| `string` | Name of role to assign |

Request example
```http
PATCH https://api.romanager.jaydensar.net/v1/role/95158139
```
```json
{
    "roleRank": 254
}
```

## Exile a user

```http
DELETE /exile/:userId
```

Request example
```http
DELETE https://api.romanager.jaydensar.net/v1/exile/95158139
```

## Set the group shout

```http
  PATCH /shout
```
*Only one of these parameters are required*
| Parameter | Type     | Description             |
| :-------- | :------- | :---------------------- |
| `content` | `string` | The new shout's content | 

Request example
```http
PATCH https://api.romanager.jaydensar.net/v1/role/95158139
```
```json
{
    "content": "This is what will be set as the new shout!"
}
```
