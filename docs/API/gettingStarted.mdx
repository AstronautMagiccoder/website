RoManager provides a REST API that allow you to perform Roblox site actions via RoManager.

Base URL: `https://api.romanager.jaydensar.net/v1`

## Generating an API key

To get an API key, run the `.api-key generate` command. You must be the group owner to run this command. A DM will be sent to you with the API key.
